Kubernetes

 

Kubernetes is a container orchestration platform, designed to run containerized workloads in production. Originally, an internal project at Google, Kubernetes (k8s) was donated to CNCF in 2014 and has since become one of the most popular platforms for running containers.

Real applications often consist of multiple containers, spread across multiple nodes. Kubernetes gives you a way to manage these containers and deploy them to the end users. Some of its features include:

  • Automatic binpacking: automatically places containers across nodes based on resource requirements and constraints
  • Horizontal scaling: easily create/delete multiple replicas of an application
  • Self healing: restarts containers that fail, container health checking
  • Service discovery and load balancing: groups application containers with a DNS name and load balances requests among them.
